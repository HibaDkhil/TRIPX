# TripX — Travel & Leisure Management Desktop Application

## Overview
TripX is a JavaFX desktop application that provides travel offers, destinations, accommodations, and transport services, with an integrated blog module where users can interact and share content.

This project was developed as part of the **PIDEV** program at **Esprit School of Engineering – Tunisia** (Academic Year **2025–2026**).

## Features
- Travel offers management (destinations, accommodations, transport)
- Community blog module (user interaction and content)
- AI Chat Assistant based on user preferences
- AI Trip Planner based on user preferences
- AI Recommendations for destinations (data-driven)
- Climate Matcher (machine learning-based)
- User Activity analytics (machine learning-based screen time/activity insights)
- Stripe payments integration
- Admin dashboard with map visualization

## Getting Started

### Prerequisites
- Java (JDK 17 )
- IDE (IntelliJ IDEA / Eclipse / NetBeans)
- MySQL + phpMyAdmin (XAMPP)
- (Optional) Stripe account and keys (if running payment flows)

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/HibaDkhil/Esprit-PIDEV-3A2-2026-TripX.git
   ```
2. Import the project into your IDE.
3. Start MySQL (via XAMPP) and create/import the database.
4. Configure database credentials in the project configuration file (add the exact file path here).
5. Run the JavaFX application.
